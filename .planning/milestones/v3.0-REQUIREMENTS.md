## v3.0 Requirements (originally v2.0 Production)

### Orçamento e Metas (BUDG)

- [x] **BUDG-01**: Usuário pode definir meta de gasto mensal por categoria
- [x] **BUDG-02**: Dashboard exibe progresso da meta com barra visual
- [x] **BUDG-03**: Alerta visual quando gasto ultrapassa 80% da meta; alerta crítico quando > 100%

### Notificações (NOTF)

- [x] **NOTF-01**: Sistema envia lembrete semanal para registrar gastos (email)
- [x] **NOTF-02**: Alerta automático quando gasto do mês ultrapassa média histórica + 20%

### Infraestrutura e Deploy (INFR)

- [x] **INFR-01**: Migração SQLite → PostgreSQL para produção
- [x] **INFR-02**: CI/CD com GitHub Actions
- [x] **INFR-03**: Deploy automatizado: backend Railway + frontend Vercel

### Assistente IA (IA)

- [x] **IA-01**: Assistente conversacional que interpreta linguagem natural para registrar gastos/receitas
- [x] **IA-02**: Contextual multi-etapas com perguntas complementares
- [x] **IA-03**: Continuidade conversacional, saudações e fim de conversa amigável
