## Wave 1: Endpoint de Dashboard

### Task 1.1 — Criar view `dashboard()`

**Files modified:** `backend/api/views.py`, `backend/api/urls.py`

**Read first:**
- `backend/api/views.py` (gastos view — padrão de filtragem por family/user)
- `backend/api/models.py` (Gasto, Family, FamilyMembership)
- `backend/api/urls.py` (padrão de registro de endpoints)

**Action:**

1. Adicionar a view `dashboard()` em `views.py`:
   - Método: `GET`
   - Query params: `mes` (1-12, obrigatório), `ano` (YYYY, obrigatório)
   - Filtrar gastos pelo mesmo padrão da view `gastos()` (family do usuário ou próprios)
   - Filtrar pelo período: `data__month=mes` e `data__year=ano`
   - Calcular e retornar JSON com:
     ```python
     {
       "periodo": {"mes": 6, "ano": 2026, "mes_nome": "Junho"},
       "total_mes": float,
       "total_mes_anterior": float,
       "variacao_absoluta": float,
       "variacao_percentual": float,
       "media_diaria": float,
       "maior_gasto": {"valor": float, "categoria": str, "descricao": str},
       "quantidade_gastos": int,
       "ranking_categorias": [
         {"categoria": str, "nome": str, "total": float, "quantidade": int, "percentual": float}
       ],
       "evolucao_12meses": [
         {"mes": 1, "ano": 2026, "total": float, "quantidade": int}
       ]
     }
     ```
   - `variacao_percentual` = `(total_mes - total_mes_anterior) / total_mes_anterior * 100` se anterior > 0, senão `0`
   - `evolucao_12meses`: últimos 12 meses completos (independente do filtro), ordenados cronologicamente
   - `ranking_categorias`: ordenado por `total` descendente, apenas categorias com gastos > 0
   - Usar Django ORM `annotate()` e `aggregate()` para eficiência

2. Registrar URL em `urls.py`:
   ```python
   path('dashboard/', views.dashboard),
   ```

**Acceptance criteria:**
- `views.py` contém `def dashboard(request):`
- `urls.py` contém `path('dashboard/', views.dashboard)`
- Request `GET /api/dashboard/?mes=6&ano=2026` retorna JSON com todas as chaves listadas
- `ranking_categorias` está ordenado do maior para o menor total
- `evolucao_12meses` tem exatamente 12 elementos
- Sem `mes` ou `ano` retorna 400 com mensagem de erro
- Usuário sem gastos retorna totais como `0.0` e arrays vazios (não 500)
